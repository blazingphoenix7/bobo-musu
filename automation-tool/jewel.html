Jewel mapper v210615



- Send edited model to server.
- Subtitutes normal mapping with bump mapping.
- Implements decal textures with diffuse & normal mapping.
- Multiple materials.
- Standalone version.
- Desktop interface.
- Multiple models each with multitexturing.
- Save button stores GLB model for consumption with the viewer.
- Keyboard commands :
i = displays this window.
t = displays the tools window.
q = toggles local / global references.
w = translate tool.
e = rotate tool.
r = scale tool.
x = toggles x axis constraint.
y = toggles y axis constraint.
z = toggles z axis constraint.
esc = removes the displayed tool from the interface.
- Mouse buttons :
left = places texture map.
middle = rotates camera.
right = pans camera.
